# Dossier images

Ce dossier contient toutes les images utilisées dans le portfolio :

- `avatar.jpg` — Photo de profil (à remplacer par votre photo)
- `og-image.png` — Image Open Graph pour les partages sur réseaux sociaux
- `projects/` — Captures d'écran des projets

> **Note :** Ajoutez vos propres images ici et référencez-les dans les pages HTML
> avec un chemin absolu depuis la racine, par exemple `/assets/images/avatar.jpg`.
